# MAGI_SYS: ERROR 404

A custom 404 error page inspired by the 1995 classic anime Neon Genesis Evangelion.

## The Concept

The MAGI are three supercomputers at the heart of NERV's operations, each embedded
with the personality of their creator, Dr. Naoko Akagi. Every critical decision requires
a consensus vote across all three units: Melchior, Balthasar, and Casper.

This 404 page treats the user's request as a query routed through the MAGI network.
The page was found. It was evaluated. CASPER-3 rejected it. The synchronization is lost,
the entry plug is blocked, and EVA-01 is going critical — all because you followed a
link that no longer exists.
